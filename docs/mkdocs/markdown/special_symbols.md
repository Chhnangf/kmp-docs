# 特殊符号

依赖模块: pymdownx.smartsymbols

| 符号          | 效果        |
| :------------ | :---------- |
| `(tm)`        | (tm)        |
| `(c)`         | (c)         |
| `(r)`         | (r)         |
| `c/o`         | c/o         |
| `+/-`         | +/-         |
| `-->`         | -->         |
| `<--`         | <--         |
| `<-->`        | <-->        |
| `=/=`         | =/=         |
| `1/4 3/5`     | 1/4 3/5     |
| `1st 2nd 3rd` | 1st 2nd 3rd |
